# HA Facturation

Logiciel de facturation desktop pour garage automobile indépendant.

## Fonctionnalités
- Création et gestion de factures
- Calcul automatique HT / TVA / TTC
- Export PDF professionnel
- Historique des factures
- Partage par e-mail
- Sauvegarde locale / cloud
- Application portable (.exe)

## Stack technique
- Python 3
- PySide6 (GUI)
- SQLite
- ReportLab (PDF)
- PyInstaller

## Philosophie
Projet conçu comme un vrai logiciel métier, 
orienté usage réel et fiabilité, 
pas comme un simple exercice académique.
